# CodeArts Bar 1.16.36

发布日期：2026-07-16

`1.16.36` 统一了 Desktop、VS Code 和 JetBrains 的会话管理体验，新增跨页多选、批量导出、项目维度筛选、每页条数选择和页码跳转，并过滤内置子任务。同步优化开发者工作台界面、跨端导出协议、聚合冷启动及 Windows 发布稳定性。

## 新增功能

- VS Code 和 JetBrains 会话列表增加原生复选框、选择本页、清空选择和跨页选择。
- Desktop、VS Code 和 JetBrains 均可将多个会话批量导出为一个 Excel、Markdown 或 JSON 文件。
- 会话与请求日志统一支持每页 `10 / 20 / 50 / 100` 条，可直接输入页码跳转。
- 分页区域显示当前记录范围、总数、当前页和总页数。
- 来源、模型和项目筛选支持多选，项目维度可用于分析和会话筛选。
- Desktop 使用分析工具栏增加项目筛选，项目范围同步作用于摘要、趋势、模型/来源统计和请求日志。

## 修复与改进

- 会话选择键改为 `source:id`，避免 Desktop 与 CLI 数据源存在相同 session ID 时选择冲突。
- 切换数据库会清空已选会话，避免将旧数据源的选择带入新数据源。
- 单会话和批量导出统一过滤 CodeArts 内置子任务，不再导出内部调度记录。
- 批量导出复用相同的规范化模型和隐私规则，只确认一次内容范围并选择一次保存位置。
- Desktop 参考 CC Switch 的原生桌面工具感，统一冷灰画布、电蓝交互强调、紧凑分段控件、键盘焦点和 disabled 对比度；标准、窄屏、宽屏、会话与日期弹层进入视觉回归。
- Electron 主 dashboard 跳过首屏未使用的 `part` 扩展性能查询；sidecar 全命中时在开库前返回，模型筛选会话使用 token 与 session sidecar 做 scoped 对账。
- VS Code 完整分析页与侧栏统一使用顶部会话导出栏，移除重复的逐行格式按钮，并压缩页头品牌与页面标题层级。
- Windows 发布目录原子重命名在遇到 `EPERM`、`EBUSY` 或 `EACCES` 时最长重试 30 秒，降低文件索引和杀毒扫描导致的打包失败。
- VS Code 与 JetBrains 插件版本同步递增，安装时可正确覆盖 `1.16.35`。

## 导出与隐私

- JSON、Markdown 和 XLSX 均来自同一规范化会话模型，不受 UI 分页或 Snapshot 样本上限影响。
- 默认脱敏凭据、用户名和本机路径，不包含推理及工具输入输出。
- 内置子任务在 Desktop、VS Code、JetBrains、单会话和批量路径中使用同一过滤规则。
- Excel 使用真实 `.xlsx` 文件，并保留中文、Emoji、长文本、请求错误类型和稳定 session ID 关联。
