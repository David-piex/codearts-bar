# CodeArts Bar 1.16.37

发布日期：2026-07-19

`1.16.37` 是面向 GitHub 主仓库发布的稳定版本，延续 `1.16.36` 的跨端会话管理能力，并集中收敛聚合性能、发布可靠性和开发者工作台视觉一致性。

## 修复与改进

- Desktop dashboard 使用 lean 聚合路径，减少首屏未消费的扩展性能查询；rollup 全命中时可在打开 SQLite 前直接返回。
- 模型筛选的会话汇总通过完整 token sidecar 限定 session ID，再与 session sidecar 合并，保持模型和时间范围语义。
- 聚合缓存、rollup 恢复、SQL.js/native 热路径和真实 fixture 测试增加回归覆盖。
- 发布目录原子替换继续对 `EPERM`、`EBUSY` 和 `EACCES` 做限时重试，降低 Windows 文件索引或杀毒扫描造成的失败。
- Desktop、VS Code 和 JetBrains 工作台统一冷灰画布、电蓝交互强调、焦点环、placeholder 和 disabled 对比度，并更新视觉基线。
- GitHub 作为主仓库，README、npm repository 元数据、Release 和 clone 指引统一指向 GitHub；Gitee 保留为同步镜像。

## 本地验证

- 重新构建 Windows 安装版、便携版、VSIX、JetBrains 插件和 CLI/npm 包。
- 完整测试、Electron E2E、VS Code E2E、聚合压力测试、视觉回归、产物体积门禁和发布产物 smoke 全部通过。
- 发布产物通过 `SHA256SUMS.txt` 校验，版本元数据统一为 `1.16.37`。
